HotelRoomReservationApp
=======================

Pre Requisites
-----------------------
* Developer Laptop Installations
* Eclipse
* Git
* Maven
* Java 7




Steps to run the application
-----------------------------
- Build the project using : mvn clean install
- Executable jar file will be created in the target folder : HotelRoomReservation-0.0.1-SNAPSHOT-executable.jar
- Execute using : java -jar target/HotelRoomReservation-0.0.1-SNAPSHOT-executable.jar Rewards:16Mar2009,17Mar2009,18Mar2009
